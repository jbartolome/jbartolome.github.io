Homework 7: SVG Files in CNIT 131A

The Scalable Vector Graphics (SVG) that I have used include a rectangle, circle, and triangle. The triangle features a gradient color fill that has been defined using the linearGradient tag within a pair of defs tags in the SVG file for the triangle. The gradient color fill is then referenced in the fill attribute of the polygon element. I have selected to use the object tags and separate SVG files to embed these three shapes. I personally like to reuse my code, so in the instance I need to reuse any of these shapes in another file, I can easily embed them using the object tags.
